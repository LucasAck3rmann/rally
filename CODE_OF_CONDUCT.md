# Código de Conduta

Este projeto adota o **Contributor Covenant, v2.1**.
Texto oficial: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Em resumo: comprometemo-nos a manter um ambiente **acolhedor, respeitoso e livre
de assédio** para todos, independente de identidade ou experiência.
Comportamentos inaceitáveis podem ser reportados a **conduta@rally.app**;
todas as queixas são analisadas de forma justa e confidencial.
